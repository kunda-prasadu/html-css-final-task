# Agency Landing Page

## 📌 Description
Responsive landing page built using HTML and CSS based on a Figma design.

## 🚀 Features
- Fully responsive (Mobile, Tablet, Desktop)
- Clean UI design
- Hover effects on buttons
- Flexbox layout

## 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox)

## ▶️ Run Project
Just open `index.html` in browser.

## 📁 Assets
All images are stored in `/assets` folder.

## ⚠️ Notes
- No JS functionality added (as per requirement)
- Focused on pixel-perfect UI
